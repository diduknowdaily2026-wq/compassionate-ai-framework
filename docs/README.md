## 🔤 Translation Notes

### Buddhist/Confucian Terms

| Chinese | English | Notes |
|---------|---------|-------|
| 慈 | Mettā | Pali term for loving-kindness |
| 悲 | Karuṇā | Compassion, desire to alleviate suffering |
| 喜 | Muditā | Empathetic joy |
| 捨 | Upekkhā | Equanimity |
| 般若 | Prajñā | Wisdom (transcendent wisdom) |
| 五蘊 | Five Aggregates | Form, feeling, perception, formations, consciousness |
| 五倫 | Five Relationships | Confucian relational ethics |
| 觀自在菩薩 | Avalokiteśvara | Bodhisattva of Compassion (Guanyin) |

### Translation Philosophy

- **Accuracy over literalism**: Buddhist/Confucian concepts are translated with conceptual accuracy rather than word-for-word
- **Pali/Sanskrit terms preserved**: Where appropriate, original Pali/Sanskrit terms are used (e.g., Mettā, Karuṇā, Prajñā)
- **Cultural context maintained**: Positionality statement acknowledges Hong Kong research context
